# 👩‍💻 About The Project

The **Aavishkar Institute Management System** is a comprehensive web-based platform developed to digitize and simplify institute management operations.

The system includes:
- Visitor Frontend
- Student Panel
- Admin Panel

It helps manage admissions, courses, student communication, leave applications, feedback, announcements, and overall institute workflow through a centralized platform.

Developed as a BCA semester project, the application focuses on combining frontend and backend technologies to create a complete real-world management system.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/TanuSharma08/aavishkar-institute.git
```

```bash
cd aavishkar-institute
```

---

# ⚙️ Setup Instructions

## 1️⃣ Move Project to Server Directory

### XAMPP

```text
C:\xampp\htdocs\
```

### WAMP

```text
C:\wamp\www\
```

---

## 2️⃣ Import Database

- Open `phpMyAdmin`
- Create a database named:

```text
aavishkar_institute
```

- Import the SQL file from:

```text
database/
```

---

## 3️⃣ Configure Database Connection

Update database credentials if needed:

```php
$cn = mysqli_connect("localhost", "root", "", "aavishkar_institute");
```

---

## 4️⃣ Run the Application

Open your browser and navigate to:

```text
http://localhost/aavishkar-institute/index.php
```
